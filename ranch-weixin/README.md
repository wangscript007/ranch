## 微信

微信模块主要实现微信接口适配。

属性

|属性|类型|说明|
|---|---|---|
|key|char(100)|引用key，不可重复。|
|name|char(100)|名称。|
|appId|char(100)|APP ID，不可重复。|
|secret|char(100)|密钥。|
|token|char(100)|验证Token。|
|mchId|char(100)|商户ID。|
|mchKey|char(100)|商户密钥。|
|accessToken|char(100)|当前Access Token。|
|jsapiTicket|char(100)|当前Jsapi Ticket。|
|time|timestamp|更新时间。|
